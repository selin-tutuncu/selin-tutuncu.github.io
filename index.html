    • Verilen benzer bir düzenle basit bir web sayfası oluşturun. CSS'in temel özelliklerini (display, width, height, margin, padding, border, selector) uygulayın. Sadece HTML5 + CSS3 kullanın, hazır framework kullanmak yasak.

    İçerik

    Bu alanda örnek içerik metinleri bulunmaktadır. CSS özellikleriyle bu metinler arasında boşluklar (line-height) ve hizalamalar (text-align) düzenleriz.

    Önce CSS ile kutuları yan yana getirmeyi, ardından alt alta yerleştirmeyi öğrendik. Şimdi ise CSS seçicilerini kullanarak grup ve hiyerarşik seçimler yapmayı göreceğiz.

    Yan Menü / Bilgi

    Bu alan yan menü veya bilgi kısmıdır. CSS ile genişlik (width), yükseklik (height), margin/padding ve border kullanarak düzenleriz.

    Öğrendiklerimizi hiyerarşik ve grup seçicilerle uygulayıp farklı bölümlere stil verebiliriz.
